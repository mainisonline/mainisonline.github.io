Techniques

C code running on Raspberry Pi. Each person created their unique program which runs a real-time audio processor using the JACK Audio Connection Kit audio server.

The output of my code creates a delay effect and frequency modulation. The delay effect creates an echo-like effect by combining the input audio with previously delayed samples. Frequency modulation alters tone using a LFO, which is applied to frequency dynamically.
